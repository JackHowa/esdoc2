    esdoc2

    esdoc2 is a documentation generator for JavaScript.
    Please try it out!

    Features

    • Generates good documentation.
    • Measures documentation coverage.
    • Integrate test codes into documentation.
    • Integrate manual into documentation.
    • Parse ECMAScript proposals.
    • esdoc2 Hosting Service

    Users

    • esdoc2 (self-hosting 😄)
    • RxJS
    • Sketch API

    And more.

    Quick Start

    # Move to a your project directory.
cd your-project/

# Install esdoc2 and standard plugin.
npm install esdoc2 esdoc2-standard-plugin

# Create a configuration file.
echo '{
  "source": "./src",
  "destination": "./docs",
  "plugins": [{"name": "esdoc2-standard-plugin"}]
}' > .esdoc2.json

# Run esdoc2.
./node_modules/.bin/esdoc2

# View a documentation
open ./docs/index.html
